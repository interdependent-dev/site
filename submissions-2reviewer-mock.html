Interdependent · Submissions 2-Reviewer Concept
Evaluation

Each script can now be read by two independent analysts. Casey returns a single weighted craft score (0–100). Baraka returns a craft score and a separate Championability rating (would an exec advocate?), combined through a decision matrix. Toggle a reviewer, or compare them side by side.

Past Submissions

Two reviewer columns. Casey has read the full back catalogue; Baraka is new, so most rows show Run Baraka until a second read is requested. When both have read a script, agreement and splits surface at a glance.
